ReadMe File for Facial Recognition-based Attendance System
####################################################################

- Extract all the files 
- The folder should contain 
	- images folder
	- attendance.py python script
	- attendance.csv file(to mark attendance)

- Capture student images to images folder.
- Name the files according to  the student's name.

- To run the python script, two external modules are required
	- OpenCV and face_recognition modules are needed to be installed
	- Run the following commands to install the modules
		- $ pip3 install opencv-python
		  $ pip3 install face-recognition

- Run the script using following command
	- $ python3 attendance.py

- Whenever the face is matched with any one in the images folder
  the attendance is marked to attendance.csv file
